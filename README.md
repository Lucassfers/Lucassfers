### My Task

**Kanban-style task management system**, created to organize boards, lists, tasks and comments in a simple and productive way.

I developed features in both the **front-end** and the **back-end**, working with interface development, API integration, authentication, database modeling, business rules and dashboard data.

🔹 **Front-end:** React + TypeScript + Vite + TailwindCSS  
🔹 **Back-end:** Node.js + Express + TypeScript + Prisma + PostgreSQL  
🔹 **Tools and libraries:** JWT, Bcrypt, Zod, Zustand, React Router, React Hook Form, Sonner, dnd-kit, Victory, ESLint  

**Repositories:**

<a href="https://github.com/Lucassfers/My-task-front-end" target="_blank">
  <img src="https://img.shields.io/badge/Front--end-181717?logo=github&logoColor=fff&style=for-the-badge" />
</a>
<a href="https://github.com/Lucassfers/My-task-back-end" target="_blank">
  <img src="https://img.shields.io/badge/Back--end-181717?logo=github&logoColor=fff&style=for-the-badge" />
</a>

**Main features:**

- User registration and login
- Secure authentication with JWT
- Password encryption with Bcrypt
- Password validation with minimum security rules
- Protected routes using token middleware
- Board creation, listing, editing and deletion
- Board categories by purpose: work, study, personal and other
- List creation, listing, editing and deletion
- Task creation, listing, editing and deletion
- Task deadline control
- Task highlight system
- Separation between highlighted and non-highlighted tasks
- Comments linked to tasks
- Search by board, list and task
- Kanban-style organization
- Drag-and-drop support on the front-end
- User-specific boards and data filtering
- Responsive interface
- Toast notifications
- Form handling with React Hook Form
- Global state management with Zustand
- PostgreSQL database modeling with Prisma
- Deploy configuration for Vercel

### ClassHub

**Marketplace platform that connects students and teachers for private classes.**  
Developed as an MVP for the Systems Analysis and Development integrator project at UniSenac Pelotas.

I participated in the development of both **front-end** and **back-end** features, contributing to the user interface, API integration, business rules and database structure.

🔹 **Front-end:** React + TypeScript + Vite + TailwindCSS  
🔹 **Back-end:** Node.js + Express + TypeScript + Prisma + PostgreSQL  
🔹 **Tools and libraries:** JWT, Bcrypt, Zod, Zustand, React Hook Form, React Router, Sonner, Flowbite, React Big Calendar, Nodemailer, ICS  

**Main features:**

- Registration flow for students and teachers
- Login system with JWT authentication
- Password encryption with Bcrypt
- User profile management
- Student and teacher-specific dashboards
- Teacher profile page with description, education, modality, hourly rate and reviews
- Subject and category listing
- Teacher search by subject
- Public teacher profile with available class schedules
- Teacher agenda management
- Student agenda view
- Class scheduling flow
- Payment simulation with PIX, credit card and debit card options
- Automatic creation of class records after booking
- Payment record creation linked to scheduled classes
- Class confirmation by student and teacher
- Class status control: waiting, confirmed, canceled and completed
- Email notification for class booking and confirmation
- Calendar event generation with ICS file
- Class history for students and teachers
- Review system for students to rate teachers
- Teacher reputation through rating and comments
- Support page
- API routes for users, students, teachers, addresses, categories, subjects, education records, schedules, classes, payments, logs and reviews
- Database modeling with relationships between users, students, teachers, classes, schedules, payments and reviews
